# FAT12 lib

## How to make a demo FAT12 imange

* install mtools
  * macOS: `brew install mtools`
  * Linux:: `sudo apt install mtools`
* create a 720KB demo FAT12 image

```sh
make fat12
```

## How to run a test

```sh
make test
```

## How to run a demo

```sh
make run
```
